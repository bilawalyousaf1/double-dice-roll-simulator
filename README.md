# Double Dice Roll Simulator 🎲

A simple and interactive web-based application that simulates rolling two dice with animations and sound effects.

## 🚀 Features
- Roll two dice with a single click
- Random number generation (1–6)
- Smooth animations using Animate.css
- Sound effect on each roll
- Responsive design with modern UI (glassmorphism + gradient)

## 🛠️ Built With
- HTML
- CSS
- JavaScript (Vanilla)
- Font Awesome
- Animate.css
- Google Fonts

## 📂 Project Structure
- index.html → Structure
- style.css → Styling
- script.js → Logic
- audio/ → Sound effects

## 📚 What I Learned
- DOM manipulation & event handling
- Random number generation in JavaScript
- CSS animations, transitions & responsive design
- Integrating external libraries via CDN
- Adding audio in web applications

## ▶️ How to Run
Simply open `index.html` in your browser.
